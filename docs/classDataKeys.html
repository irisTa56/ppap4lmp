Detailed Description

DataKeys is a small class owning a set of string keys in Element::data.

An object of this class is owned by an Element object and used for checking existence of keys for properties in the Element::data quickly. Note that checking keys against the Element::data itself is a time consuming process if it is a large array Json object.
