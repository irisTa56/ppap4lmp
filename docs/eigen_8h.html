using ArrayXXd = Eigen::Array<double, Eigen::Dynamic, Eigen::Dynamic, Eigen::RowMajor>

ArrayXXd is an alias for a two-dimensional array of Eigen where the values are double.

Although ArrayXXd is column major by default, in this program, row major is preferable so that the array can be passed to Python as Numpy-Array, which is row major, without confusing.
